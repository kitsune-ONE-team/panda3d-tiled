Tiled importer for Panda3D
==========================

Python library for loading [Tiled](https://www.mapeditor.org/) maps into [Panda3D](https://www.panda3d.org/).


Features
--------

* can load spritesheet based tilemaps
* can load tiles from separate image files
* supports tile layers
* supports object layers


Requirements
------------

* [pytiled_parser](https://github.com/pythonarcade/pytiled_parser)
* [Panda3D](https://www.panda3d.org/)
